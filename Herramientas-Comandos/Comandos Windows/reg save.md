#regsave

---

Guarda una copia de las subclaves, entradas y valores especificados del Registro en un archivo especificado.

---
# Comando

```shell
reg save Archivo DirectorioDondeQuieroGuardarElArchivo
Ejemplo:reg save hklm\sam c:\Temp\sam
```
